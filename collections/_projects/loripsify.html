Loripsify

Blog posts aren't static and uniform - so your Lorem Ipsum shouldn't be either!

How to use Loripsify?

  • Scroll over Loripsify's handle
  • Select desired formatting options
  • Adjust paragraph density sliders
  • Input a container or use the default
  • Generate and preview your document!

Lorip-si-who?

Loripsify is a combination of Lorem Ipsum and the suffix -ify. Literally: "Make Lorem Ipsum"

What is Loripsify?

Loripsify is a tool designed to simplify your design workflow. Effortlessly showcase beautiful, elegant layouts to your clients or simply refine your own designs by introducing rich formatting.

Why use Loripsify?

As complexity increases, it quickly becomes impractical to manually insert chunks of text within every individual container. Moreover, using the same static passage repeatedly fails to capture the true essence of your design's variability.

Sourcing your own documents manually is a challenging endeavor in itself, let alone incorporating a method to seamlessly display them in the context of your workflow.

How does Loripsify work?

When you click generate, a fetch request with the appropriate endpoints is wrapped by the specified proxy and sent to loripsum.net/api. The return value is injected into the specified container using innerHTML, or a newly created container if one was not given.

Proxy

Loripsum does not allow Cross Origin Resource Sharing, so a proxy is required to bypass this restriction. The best and safest option would be to use your own server to wrap the request but a default option has been provided for convenience.

! WARNING !

DO NOT USE A PUBLIC PROXY WHEN WORKING WITH SENSITIVE DATA

A malicious actor in control of the proxy could respond to your request with an iframe or other HTML (or worse) capable of executing code locally in your environment. Loripsify will sandbox all containers before setting innerHTML in attempts to mitigate this kind of attack.
